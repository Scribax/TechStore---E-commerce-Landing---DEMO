# 🛒 TechStore - E-commerce Landing Page

## 📋 Descripción del Proyecto

TechStore es una **landing page de e-commerce moderna** especializada en tecnología, desarrollada como proyecto de demostración para mostrar habilidades en desarrollo web frontend. El proyecto implementa una experiencia de usuario completa con carrito de compras, filtros dinámicos, animaciones avanzadas y diseño completamente responsive.

---

## 🚀 Instalación y Configuración

### **Prerrequisitos**
- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Servidor web local (opcional para desarrollo)

### **Instalación Rápida**

1. **Clonar o descargar** el proyecto:
```bash
git clone https://github.com/tu-usuario/techstore-demo.git
cd techstore-demo
```

2. **Abrir en navegador**:
   - Opción 1: Abrir `index.html` directamente
   - Opción 2: Usar servidor local (recomendado)

### **Servidor Local (Recomendado)**

```bash
# Con Python 3
python -m http.server 8000

# Con Node.js (npx)
npx serve .

# Con PHP
php -S localhost:8000
```

Luego abrir: `http://localhost:8000`

```javascript

    badge: 'New',
    specs: ['Spec 1', 'Spec 2'],
```

### **Modificar Animaciones**
```css
/* Ajustar velocidad de transiciones */
:root {
    --transition-fast: 0.15s ease;
    --transition-normal: 0.3s ease;
    --transition-slow: 0.5s ease;
}
```
